# MY-Trade-Journal UI Prototype

Manual-execution trading intelligence UI prototype.

## Web app location

The active Next.js prototype lives in:

- `apps/web`

## Run locally

```bash
cd apps/web
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scope guardrails

- Mock data only (no live Supabase integration yet)
- No broker execution
- No auto-trading
- No Apps Script / clasp / Google Sheets logic
- No secrets in source
